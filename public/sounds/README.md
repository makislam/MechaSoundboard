# Sound Files

Place your `.mp3` files in this directory.
The filenames should match the IDs used in `App.jsx`:

- `welcome.mp3`
- `gong-cha.mp3`
- `judging.mp3`
- `open-d1.mp3` (Opening Day 1)
- `close-d1.mp3` (Closing Day 1)
- `open-d2.mp3` (Opening Day 2)
- `close-d2.mp3` (Closing Day 2)
- `open-d3.mp3` (Opening Day 3)
- `close-d3.mp3` (Closing Day 3)
